# Study helper
Web aplication with different types of mode to help studying
- flashcards
- connect meaning

Questions can be added by prisma / localhost the web application with npm or pnpm (when I get to them).

## node packages
 - Vite
 - Tailwind
 - react-router-dom
 
## server node packages
 - Prisma
 - Express
